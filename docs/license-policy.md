# License Policy

FadeOut uses a proprietary, all-rights-reserved license.

## Decision

Do not use an open-source license for this codebase.

Chosen policy:

```text
FadeOut Proprietary License
Copyright (c) 2026 HelaGen. All rights reserved.
```

## Why

The product goal is to prevent others from copying, modifying, distributing, or
commercializing the code. Open-source licenses such as MIT, Apache, BSD, and GPL
grant reuse rights, so they do not match that goal.

## Practical Meaning

People may view the code if they can access the repository, but they are not
granted permission to:

- Copy the code.
- Reuse the code in another app.
- Modify and redistribute the code.
- Publish forks outside platform terms.
- Sell or commercialize the code.
- Create derivative products from the code.

## Before Launch

Have a lawyer review `LICENSE` before publishing the repository publicly or
before accepting external contributors.

If outside contributors are ever allowed, use a contributor agreement or written
assignment so HelaGen keeps clear ownership of the code.
